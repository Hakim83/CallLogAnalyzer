PRIVACY POLICY FOR CALLS ANALYZER APPLICATION

This privacy policy governs your use of the software application  (“Calls Analyzer”) for mobile devices that was created by Mohammed Abdalhakam. 

What is the purpose of the application?

The Application is a tool to analyze calls in a time period specified by the user and gives back calls info in this period like number of calls, most called contacts, period of calls, types of numbers (mobile, landline ..etc.) and geographical locations of them. The application is open source

 What information does the application access and how the application access it?

When you select a time period to analyze, the application needs to get permission for reading call log by requesting to be a default phone dialer.

How the application uses collected information?

The application uses calls information to display analysis result as mentioned in the application purpose.

Do third parties see and/or have access to information obtained by the Application?

No, the collected calls information is used in the application without being transport to any other party. Neither application owner nor any other party can access these information.

What are my opt-out rights?

You can stop all collection of information by the Application easily by uninstalling the Application. You may use the standard uninstall processes as may be available as part of your mobile device or via the mobile application marketplace or network. 

Changes
